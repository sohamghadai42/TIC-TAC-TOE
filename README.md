# TIC-TAC-TOE
A simple tic tac toe game built with HTML, CSS, and JS.
You can play with your friend(Player X vs Player O) on a 3x3 grid.
## How to play
- Open 'index.html' in your browser.
- Player X goes first.
- Take turns until someone wins or the game is a draw.
## Technologies
- HTML
- CSS
- JS
